AurisBridge

An open-source, ultra-low latency Android app designed to turn everyday smartphones into powerful live-listen accessibility tools.

View on GitHub

Why Aurisbridge?

Standard Bluetooth audio creates a disorienting echo for hearing aid apps due to high latency. Aurisbridge bypasses standard audio paths using C++ native libraries to deliver real-time sound to True Wireless Stereo (TWS) earbuds.

⚡ Zero Echo

Powered by Google's Oboe C++ library, achieving sub-30ms round-trip latency to eliminate the "delayed auditory feedback" effect.

🎛️ Clarity Boost

Custom digital signal processing (DSP) biquad filters specifically designed to boost the high frequencies (2kHz - 8kHz) commonly lost in age-related hearing loss.

📱 Senior-Friendly UI

Designed with extreme accessibility in mind: massive touch targets, high-contrast colors, and zero complicated equalizer sliders.

Built with care. This is an open-source accessibility project and does not replace medical hearing aids.
